# Golang Restful API implementation
Here I used below technologies
- Gin framework
- Swagger (Swaggo)
- SQLite
- Testify

## How to run
Before running you can download dependencies by:
```console
foo@bar:~$ go get .
```
Then generate swagger docs by

```console
foo@bar:~$ swag init
```
Then simply run the application

```console
foo@bar:~$ go run main.go
```
